# Indústria Conectada

Sistema de Monitoramento e Gestão Industrial desenvolvido como Projeto Integrador do Curso Técnico em Informática para Internet.

## Sobre o Projeto

O Indústria Conectada é uma aplicação web desenvolvida para centralizar e gerenciar informações relacionadas ao ambiente industrial. O sistema permite o cadastro e monitoramento de máquinas, acompanhamento da produção e visualização de indicadores importantes para auxiliar na tomada de decisões.

A proposta surgiu da necessidade de substituir controles realizados em planilhas e documentos separados, oferecendo uma solução integrada, organizada e de fácil utilização.

## Objetivo

Desenvolver uma aplicação Full Stack para monitoramento e gestão de processos industriais, aplicando conceitos de Desenvolvimento Web, Banco de Dados, APIs REST, Versionamento e Indústria 4.0.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Node.js
* Express
* MySQL
* MySQL Workbench
* Git e GitHub

## Principais Funcionalidades

* Cadastro de máquinas;
* Consulta de máquinas;
* Atualização de registros;
* Exclusão de máquinas;
* Dashboard com indicadores;
* Controle de produção;
* Registro de ocorrências;
* Interface responsiva.

## Arquitetura do Sistema

Usuário → Front-End (HTML, CSS e JavaScript) → API REST (Node.js + Express) → Banco de Dados (MySQL)
